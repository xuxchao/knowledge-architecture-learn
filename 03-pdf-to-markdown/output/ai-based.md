# AI 知识库构建指南

## 一、概述

本文档是 `knowledge-architecture` 项目的示例 PDF，用于演示 PDF 文档解析的两种方法。页面包含可提取的文字内容，后续页面还包含嵌入的图片和矢量图形。

## 二、知识库核心流程

构建 AI 知识库通常包括以下步骤：

1. **文档收集** — 收集各类格式的文档（PDF、Word、Markdown 等）  
2. **文档解析** — 将文档内容提取为纯文本  
3. **文本分块** — 将长文本切分为语义连贯的片段  
4. **向量化** — 使用 Embedding 模型将文本片段转为向量  
5. **存储与索引** — 将向量存入向量数据库  
6. **检索与生成** — 基于用户问题检索相关片段，交给 LLM 生成回答  

> （本页为纯文字内容，可被 `pdf-parse` 和 LangChain `PDFLoader` 提取）
